# 016 2022-07-18

## App Store Connect API 2.0 リリース

App Store Connect API 2.0 が利用可能になり、以下が更新されました。

- アプリ内購入
- サブスクリプション
- カスタマーレビューのサポート
- 電力とパフォーマンスの改善
